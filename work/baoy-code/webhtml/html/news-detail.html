“旅游养老意外保险”为50岁以上老年人出行添保障

2016年10月30日08:53 来源：央广网 作者：刘丽

为解决老年人出行难题多、意外伤害保险需求大、意外险种保障不充分、高龄老人投保困难等问题，中国老年学学会、中国老年学会老年旅游专业委员会特委托金诚国际保险经纪公司对北京、上海、广州、深圳、成都、安徽、杭州、黑龙江、鄂尔多斯等全国17家养老基地

为解决老年人出行难题多、意外伤害保险需求大、意外险种保障不充分、高龄老人投保困难等问题，中国老学会、中国老年学会老年旅游专业委京、上海、广州、深圳、成都、安徽、杭州、黑龙江、鄂尔多斯等全国17家养老基地

旅游保险的重要性

这款产品充分考虑到了老年人的各种特定风险需求，由现有多个意外险及附加险种条款进行组合，扩大了意外身故、伤残、疾病猝死、境外保障、医疗费用多种保险责任赔付范围，涵盖老年人在生活的各种场所，包括在居家生活、乘坐公共交通工具、参加公共场所活动、入住养老服务机构、外出旅游时发生的各种意外伤害事故保障，具有高保额、多保障、低保费等特点和充分的适老化、灵活性、实用性和保障性
